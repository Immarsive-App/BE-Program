## Setup

* run go mod
    ```
    go mod init projectname
    ```

* download echo
    ```
    go get -u github.com/labstack/echo/v4
    ```

* download gorm
    ```
    go get -u gorm.io/gorm
    go get -u gorm.io/driver/mysql
    ```

* download viper (to load .env automatically)
    ```
    go get github.com/spf13/viper
    ```
